# Architecture Decision Records (ADR's)

## Wat is een ADR?
Een ADR documenteert een belangrijke architectuur beslissing, inclusief de context, de beslissing zelf, en de gevolgen.

## ADR Template

Elke ADR volgt dit format:

- **Titel**: Korte omschrijving van de beslissing
- **Status**: Concept / Geaccepteerd / Vervangen
- **Context**: Wat is het probleem?
- **Beslissing**: Wat hebben we besloten?
- **Gevolgen**: Wat zijn de voor- en nadelen?
- **Alternatieven**: Wat hebben we overwogen?
