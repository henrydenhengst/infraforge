# Terminologie

| Term | Betekenis |
|------|-----------|
| Role | Herbruikbare set van taken (bv. docker role) |
| Playbook | Verzameling van roles (bv. site.yml) |
| Inventory | Lijst van servers en groepen |
| Group_vars | Variabelen per groep servers |
| Host_vars | Variabelen per individuele server |
| Task | Een enkele actie (bv. package installatie) |
| Handler | Taak die alleen bij wijziging wordt uitgevoerd |
| Tag | Label voor gerichte uitvoering |
| Molecule | Test framework voor Ansible roles |
| Ansible Vault | Encryptie voor secrets |
